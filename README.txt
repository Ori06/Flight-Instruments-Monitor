A project by Ori Ofer

How to Run?

1. Clone the repository:

git clone https://github.com/your-username/flight-instruments-monitor.git
cd flight-monitor

2. Install dependencies:

npm install

3. Set up your .env file n the root folder with the following:

MONGODB_URI=mongodb://localhost:27017/flightInstrumentsMonitor
PORT=3000

4. Start the backend:
cd server
npm install
npm run server

5. Start the frontend:
cd ..
npm run dev
